SVGs

Requirements

  • Gulp task(s) to compile individual SVGs into a single SVG manifest file
  • SVG manifest file must be able to be referenced externally
  • SVGs must be able to be stylable, while retaining their original detail
    • Interaction effects (hover, click, etc.)
    • Animation

Externally Referenced SVG

<svg class="icon icon--lightbulb">
    <use xlink:href="svg/symbols.svg#lightbulb_illustrator"></use>
</svg>
